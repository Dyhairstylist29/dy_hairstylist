# DY Hairstylist – Luxury Hair Studio Website

<img width="1285" height="628" alt="image" src="https://github.com/user-attachments/assets/637a81cb-c49e-4046-9160-d39fddc327f1" />


A modern luxury hairstylist website designed and developed to elevate the online presence of **DY Hairstylist**.

Built with a strong focus on elegance, premium user experience, and seamless appointment booking.

---

## Overview

The website was created to provide clients with a premium digital experience that reflects the quality of DY Hairstylist's work.

Rather than functioning as a simple portfolio, the website guides visitors through a luxurious booking journey—from discovering signature hairstyles to reserving an appointment.

---

## Features

- Elegant soft luxury UI
- Fully responsive design
- Light & Dark Mode
- Animated Hero Section
- Signature Styles video gallery
- Premium service showcase
- Smooth scroll animations
- Luxury booking experience
- Studio Experience & Home Experience selection
- Dynamic booking form
- Booking summary
- WhatsApp appointment integration
- Mobile-friendly navigation
